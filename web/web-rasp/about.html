RASPURI v2

About

RASPURI v2 is a new developed RASP (Regional Atmospheric Soaring Program). The idea came from updating to a 64 bit architecture and WRF to v4 the original RASPURI, which is a original Dr Jack's RASP. RASPURI v2 is full coded in Python and Bash, except the Fortran subroutines developed by Dr. Jack for some of the postprocessings. The whole application runs through Docker.

Project

The project can be found in the Github repository: oriolcervello/raspuri.
